Внешняя обработка "ИтерационныеМетоды_Суханов_МО.epf" предназначена для решения СЛАУ итерационными методами Якоби, Зейделя и релаксации. Включает в себя вспомогательные алгоритмы для предварительного анализа сходимости итерационного процесса: QR-алгоритм, метод Гивенса, алгоритм проверки возможности обеспечения диагонального преобладания основной матрицы СЛАУ, сортировка Хоара, алгоритм устранения нулевых элементов на главной диагонали основной матрицы СЛАУ, алгоритм изменения спектрального радиуса.

Количество полей ввода коэффициентов СЛАУ зависит от порядка основной матрицы системы, поэтому нижняя часть формы (после кнопки "Ввести систему") формируется программно.
