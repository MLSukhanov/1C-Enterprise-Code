Внешняя обработка для решения СЛАУ итерационными методами Якоби, Зейделя, релаксации. Включает в себя вспомогательные алгоритмы для предварительного анализа сходимости итерационного процесса: QR-алгоритм, метод Гивенса, алгоритм проверки возможности обеспечения диагонального преобладания основной матрицы СЛАУ, сортировка Хоара, алгоритм устранения нулевых элементов на главной диагонали основной матрицы СЛАУ, алгоритм изменения спектрального радиуса.
